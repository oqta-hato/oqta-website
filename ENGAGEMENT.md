# OQTA 関わりしろ設計 v0（2026-05-29）

> レンズ: [[BRAND_FOUNDATION.md]] / 奉納の文法: OQTA Brain `business/shrine-model`。
> 対象: oqta.com＝**OQTAサービス専用サイト**（INORI企業サイトは別立て）。

## 0. 関わりしろの正体 ＝「物語の登場人物になる」
OQTAはいずれ**漫画になる物語**。関わるとは、**主人公の横にいる登場人物になる**こと。顧客ではなく登場人物。＝エヴァンジェリスト／推し／奉納者／氏子・檀家。**すべての扉は「登場人物になる一手」**として設計する。招待コピーは「買ってください」でなく **「あなたも、この物語の登場人物に。」**

## 1. サイトの役割 ＝「深度を上げる場」（新規獲得ではない）
来訪者は**半分知っている人**（口コミ/展示会で見た「なんとなく面白そう」「少し分かるがまだ腑に落ちてない」）。何も知らない人はここに来ない。
→ 役割は **「ああ、そういうことか」と理解を深め、証明（1.8M回鳴動カウンター/お祈りマップ/ストーリー）で興味と所属の深度を上げる**こと。＝コールドファネルでなく**深度ファネル**。

## 2. positioning ＝ 孤高・コノサー（知られないことが価値）
- 普通には買えない／**口コミでしか辿り着けない**／**広告枠・インフルエンサーを使わない**。無名性そのものが価値。
- 「よくここまで辿り着きましたね」を**口に出さず醸す“孤高のページ”**。めちゃくちゃマニアックなファッションの佇まい。
- **煽りCTA／ポップアップ／“今すぐ購入”／SEO釣り は使わない**（抑制＝余白）。購入導線も控えめ（声高なECにしない・“ご縁があれば”の温度）。
- 成長は**拡散でなく伝播**：濃い同心円が広がる**コノサー/カルト型**。勧進帳の所属＝文化資本＝静かな口コミ動機。

## 3. 関わりしろの階段（低→深・すべて「登場人物になる」）
| 段 | 関わり方 | 受け皿 |
|---|---|---|
| 観る | お祈りマップを眺める／ストーリーを読む・Radioを聴く | inoree.world(OQTA機能として組込) / note / Heart Poppo Radio |
| 語る | 自分の物語を寄せる（→Radio取材） | Heart Poppo Radio |
| 奉納 | お賽銭・note投げ銭・Amazonアフィリエイト | コングラント / note / Amazon |
| 体験 | ライトに“親とこの体験” | **Androidレシーバー**（§5） |
| 贈る/買う | Heart Poppo を買う・贈る | shop（控えめに） |
| 続ける | 継続して支える（氏子） | コングラント継続 |
| 名を連ねる | **勧進帳/提灯に名が残る** | §4 |

## 4. 目玉 ＝ 年度版「勧進帳／ご芳名板／提灯」
- 寄進者の連なり＝聖なる営みへの参加者の記録。**年度スタンプ**（2026年度版/2027年度版…）で**「いつ物語に登場したか」を刻む**。
- **無名のうちに推す人ほど純度が高い＝最も誇り高い登場人物**。2026の提灯は時間とともに“誇り”が増す。
- ⚠️**死守する一線**：価値は**純度・誇り**であって**「早く入れば得（ROI/クラウト/投機）」ではない**（神社モデル第5条＝名は奉納の“結果として”残る順番）。NFT早期ミント的ハイプの逆。ここがブレると孤高が安っぽくなる。
- **将来サイトのメインコンテンツになりうる**（「鳩時計を持っていなくても、この文化を推す自分が嬉しい」の受け皿）。

## 5. 体験の入口 ＝ Androidレシーバー（¥30,000の壁の解消）
- Heart Poppo ¥30,000 は「パッと財布から出す」には大きい＝最大の壁。
- **中古Android/古い端末でライトに“親とこの体験”**ができるのがレシーバーの役割。広がれば体験者増＋送り手のお賽銭でインカムも増、ユーザー増＝口コミ機会増。
- **京都の鳩を鳴らすQRは役割を終える方向**（お祈りマップが太平洋を越える体験を海外/SXSWで見せるデモだった）→ **過渡的・sunset**。主役はレシーバーへ。

## 6. 深いコンバージョン ＝「語り始める／推したくなる」
真のエヴァンジェリストの瞬間＝**人にその価値を言いたくなる**こと。語りが人ごとに違うのが面白く、**それが Heart Poppo Radio のネタ源**＝コンテンツが尽きない循環。

## 7. 扉の先（まず）
くぐった人にはまず **Heart Poppo Radio を再生／note を読む**。（one of them・展開は多数）
